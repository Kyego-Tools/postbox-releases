# PostBox v0.1.0 — Initial Release 🎉

We're releasing the first public version of PostBox — a free, self-hosted social media scheduler built as an open alternative to Buffer, Hootsuite, and Later.

---

## ✨ What's included

### Scheduling
- Schedule posts to **Facebook, Instagram, TikTok, Twitter/X, and Threads** from a single dashboard
- Image, video, and text post support
- Per-platform character limit enforcement
- Drafts with scheduled date/time

### Team Collaboration
- **Role-based access**: admin, full access, approval required
- **Approval workflow** — restricted users submit for approval; posts only go live after an admin approves
- Expired-schedule detection on approval (post now or pick a new time)
- Invite team members by email with auto-generated passwords

### Analytics
- Per-platform post insights (views, reach, likes, comments, shares)
- Posting streak tracker — includes posts made natively outside the app
- Mini calendar showing all scheduled and published activity

### Media
- Upload images and videos directly to Convex storage
- Reuse existing media from drafts

### Dashboard
- Overview with upcoming posts, recent activity, streak stats
- All Posts page with filters by platform, time range, and status
- Scheduled, Drafts, and Pending Approvals in a unified tabbed view

### Self-hosting
- **One-command Docker deploy** — `docker compose up --build -d`
- First run automatically deploys Convex backend, generates auth keys, and pushes platform credentials
- Settings → About page shows current version and update availability

---

## 🚀 Getting started

**Requirements:** Docker + a free [Convex](https://convex.dev) account

```bash
git clone https://github.com/YOUR_ORG/postbox.git
cd postbox
cp .env.docker.example .env.docker
# Fill in NEXT_PUBLIC_CONVEX_URL, NEXT_PUBLIC_CONVEX_SITE_URL, CONVEX_DEPLOY_KEY
docker compose up --build -d
```

Open `http://localhost:3000` — the first account you create becomes the admin.
